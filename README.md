## 💻 Executando localmente

Clone o projeto

```bash
git clone https://github.com/RodrigoMA21/portfolio.git
```

Entre na pasta

```bash
cd portfolio
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```

## 📦 Build

```bash
npm run build
```
