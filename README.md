# 📖 Sobre o projeto

Este projeto é meu portfólio pessoal, desenvolvido para apresentar minha trajetória profissional, habilidades técnicas, experiências, certificações e principais projetos.

Além de funcionar como um cartão de visitas profissional, este projeto também demonstra minhas habilidades em desenvolvimento Front-end utilizando React.

# 💻 Como executar

Clone o repositório

```bash
git clone https://github.com/RodrigoMA21/portfolio.git
```

Entre na pasta

```bash
cd portfolio
```

Instale as dependências

```bash
npm install
```

Inicie o projeto

```bash
npm run dev
```

---

# 📦 Build para produção

Gerar a versão otimizada

```bash
npm run build
```

Visualizar localmente

```bash
npm run preview
```
